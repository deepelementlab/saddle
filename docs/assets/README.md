# Saddle 示意图资源

与 OpenHarness 宣传图结构对应的 **Saddle** 矢量示意图（深色 + 青色描边风格），便于放入 README、Wiki 或演示稿。

| 文件 | 对应参考图 | 说明 |
|------|------------|------|
| [saddle-pillars-harness.svg](./saddle-pillars-harness.svg) | 「Harness = 多能力块 + 中央 Agent」 | 顶栏公式与四支柱同上；中心圆内为 **[saddle-center-steed.png](./saddle-center-steed.png)** 位图（附件形象），经 `clipPath` 裁成椭圆，外圈保留霓虹描边。部署时请 **PNG 与 SVG 同目录** 以免插图断裂。 |
| [saddle-one-command-flow.svg](./saddle-one-command-flow.svg) | 「一条命令 + 三格漫画」 | 上：**`saddle run`** 终端；中：横幅 **WHAT / HOW / outcome**；下三栏：**PARADIGM THINKS**（spec/design/develop）、**SADDLE RUNS**（四支柱合成运行时）、**NORTH STAR READY**（资料包与目标）。 |
